# Open Paws Corporate Watch — Agent Quick Reference

Tracks corporate animal welfare commitments from 10 major food companies, monitoring pledge compliance and surfacing broken promises for advocacy campaigns.

## How to Run

```bash
# Install
pip install -e ".[dev]"

# Start API server
uvicorn src.api.server:app --reload

# Start frontend
cd frontend && npm run dev
```
